# CP_multimodal
## Installation

```
conda create --name multimodal_module python==3.10

```

## Folder structure
multimodal_module/
├── README.md
├── configs/
├── data/
├── output/
├── scripts/
├── tools/
├── unsupervised/

## Usage 
Refer to the scripts folder to run the unsupervised method for changepoint detection.

Prepare data
```
sh prepare_full_data.sh
```

Inference on full data. 
```
sh prepare_full_data.sh
```
